# English-translate-to-chinese

這個程式可以將你輸入的英文轉換成中文，並且使用的是內建輸入法。
目前的缺點是必須先複製英文文字才能進行轉換，無法實現即時的呼叫程式使用。
  
程式使用「Ctrl + Space」切換輸入法，不同的系統可能會有不同的快捷鍵設定。
  
主程式為 `teste.py`，其他檔案則為使用 NSIS 打包的程式檔案。
   
## 如果要安裝，請執行 `install 英to中轉換_1.0.exe`
